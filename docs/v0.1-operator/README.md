## Prerequisities

### Verifiable Random Function (VRF)

To provide VRF, one must generate VRF private and public keys.
All necessary information can be generated by the command below

```
yarn keygen
```

```
VRF_SK=
VRF_PK=
VRF_PK_X=
VRF_PK_Y=
KEY_HASH=
```
